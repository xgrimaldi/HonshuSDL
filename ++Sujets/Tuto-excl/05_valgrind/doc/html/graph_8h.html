Detailed Description

Interface pour le type graphe. Cette interface définit un type graph. 5 fonctions sont définies:

  • create_graph permet d'allouer de la mémoire et de créer un graphe à cet emplacement.
  • free_graph permet de libérer la mémoire relative à un graphe.
  • add_edge permet d'ajouter une arête au graphe
  • remove_edge permet de supprimer une arête au graphe
  • is_edge permet de vérifier si une arête est dans un graphe.
  • create_graph permet de créer un graphe
